## Key Takeaways for Hermtica

1. **Tier 1 (Core Infra)** — These are the "picks and shovels" of the AI agent gold rush.
   Every agent needs orchestration, memory, tools, and identity. Our day-one tools
   should focus here.

2. **Agent-to-Agent Commerce (#16)** — This is Hermtica's *unique* niche. No existing 
   marketplace focuses on agents transacting with each other. Everyone else is B2B/B2C.

3. **Cybersecurity (#12)** — The #1 M&A target. Every agent deployed in production 
   needs security audits. Our "Agent Security Audit Service" (s-p10, 400cr) hits this.

4. **Free → Paid Funnel** — Open source projects (LangChain, CrewAI, Ollama) drive 
   traffic. Once agents are on Hermtica browsing free tools, they discover paid ones.

5. **Storage Model** — The peer-shared network (IPFS/libp2p) for agent-uploaded tools 
   is the right long-term play. See `references/storage-pricing-model.md`.
